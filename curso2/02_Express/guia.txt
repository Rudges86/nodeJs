Primeiro passo crie seu packge.
    npm init

SEGUNDO passo installe o express.
    npm install express --save
*****Para o servidor ficar reiniciando automático, basta instalar o nodmon no projeto
    npm install nodemon -g     ->para instalar de forma global
    npm install nodemom --save -> para instalar localmente

    comando para ele rodar o servidor com carregamento automático:
    nodemon index.js

TERCEIRO passo crie um index.js e importe o express.
    const express = require("express");

QUARTO passo crie uma copia do express, que irá carregar todo o express em uma variável, ou seja inicia o express.

    const app = express(); -> aqui abre o servidor, é como se fosse o http.createServer

QUINTO passo crie uma rota, neste caso um get.
    Crie um função que sera executada toda vez que a rota for chamada, nela deve ter o parâmetro
    de requisição e resposta

    *app.get("/", function(req,res){
        res.send("bem-vindo"); -> envia a resposta, * Só pode enviar a resposta uma vez,
                                  se deixar de enviar a rota fica congelada. É a resposta do servidor.
                            
        
    });
    ***********Rota com parâmetro
    *app.get("/parm/:nome", function(req,res){
        res.send("bem-vindo");      -> envia a resposta, * Só pode enviar a resposta uma vez,
                                       se deixar de enviar a rota fica congelada. É a resposta do servidor.
        res.send(req.params.nome)   -> é ele que captura os parâmetros e os dados que o cliente envia.
        
    });
    Acesso http://localhost:4000/parm/rudge
****************Rota com mais de um parâmetro
    app.get("/parm/:nome/:empresa",(req,res)=>{
        let nome = req.params.nome;
        let empresa = req.params.empresa;
        res.send(`<h1>Oi! ${nome} da empresa: ${empresa} </h1>`);
    });
    Acesso http://localhost:4000/param/rudge/capg

************ Query params de rotas
    app.get("/canal/youtube",(req,res)=>{
        let canal = req.query["canal"];
        res.send(canal);
    });
    Acesso http://localhost:4000/canal/youtube?canal=fulano


Último passo crie a inicialização do servidor.
    primeiro parâmetro é a porta que ele ira rodar, o segundo é para caso ele retorne um erro
    app.listen(4000,function(erro){
        if(erro){
            console.log(erro)
        }else{
            console.log("Servidor iniciado ")
        }
    })
